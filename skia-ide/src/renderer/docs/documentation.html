SKIA FORGE - Documentation

Getting Started

  • Install dependencies with npm install.
  • Build with npm run build.
  • Launch with npm start.

Explorer

  • Use File > Open Folder to load a workspace.
  • Browse the recursive file tree in the left panel.
  • Click a file to open it in the editor.

Editor

  • Monaco editor with SKIA dark styling.
  • Menu shortcuts trigger editor actions.
  • Language detection by extension for common file types.

Search

  • Switch to Search view to find text across files.
  • Use Find in Files from the Edit menu.

Agent

  • Describe a task in the Agent input.
  • Run task from view controls or menu command.

Forge

  • Control plane view for mode and governance context.
  • Displays backend module status telemetry.

Settings

  • Configure backend URL.
  • Configure auth token.

Chat Panel

  • Ask SKIA anything from the chat panel.
  • Use New Chat and Clear controls as needed.
