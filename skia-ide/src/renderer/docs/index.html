Getting Started

  • Sign in using your SKIA account credentials.
  • Open a folder from the File menu to start working.
  • Use the chat panel for SKIA assistant workflows.

Core Views

  • Explorer: Browse and open project files.
  • Search: Find content across files.
  • Agent: Run guided task workflows.
  • Forge: View control-plane status and telemetry.
  • Settings: Personal editor preferences.
