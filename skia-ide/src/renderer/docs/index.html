Getting Started

  • Install dependencies with npm install
  • Build renderer/main with npm run build
  • Start desktop app with npm start

Keyboard Shortcuts

ActionShortcut
Open FileCtrl+O
Open FolderCtrl+K Ctrl+O
SaveCtrl+S
FindCtrl+F
Toggle TerminalCtrl+`
Toggle Developer ToolsCtrl+Shift+I

Connecting a Backend

  • Open Settings and set Backend URL + Auth Token
  • Use Run > Start SKIA Backend to launch local backend
  • Status bar shows connection state in real time
